# Friend

### Friend Function

> **def**:朋友函數雖然不是類別中的成員，卻能夠存取類別中的任何資料與函數                     &#x20;

### Friend Class

### Rules about Friend Class

* 朋友關係是單向：類別B是A的朋友，因此B可以存取A內部的資料與函數；但A不是B的朋友，因此A不能直接存取B內部的資料與函數
* 朋友關係不能轉移：
* 朋友關係不能媒合
* 朋友關係不能被繼承
